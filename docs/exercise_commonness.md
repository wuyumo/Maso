## Why this exists

We have **979** exercises bundled but no real usage telemetry. This audit estimates how recognizable each exercise is to a general gym-goer based on:
- whether the name matches a canonical lift (bench press, squat, pull-up, …)
- equipment popularity (mainstream / common / niche)
- has a demo video (proxy for content quality / familiarity)
- category bias (strength / calisthenics / cardio score above mobility / stretching)
- penalties for telltale-niche name cues ("concentration", "21s", "Yates", …)
